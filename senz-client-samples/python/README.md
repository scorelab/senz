# Sample Client : Transfer Image (python)

## Instructions
1. Install required packages
    ```
    $ pip install -r requirements.txt
    ```

2. Setup and run [Senz server](/../../senz-server/README.md)

3. Run receiver device
    ```
    $ cd device_2
    $ python3 receiver.py
    ```
    
     
4. Run sender device
    ```
    $ cd device_1
    $ python3 sender.py
    ```
