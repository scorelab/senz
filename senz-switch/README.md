## Setting up the project

### Pre-requisites

-   Scala 2.12.6 or higher
-   MongoDB Community edition (Server)

## Setting up the project using shell script

```bash
sudo bash serverInstallDebian.sh
```

## Run server

1.  `cd` in to the project folder
2.  Run `sbt run`

## Run tests

1.  `cd` in to the project folder
2.  Run `sbt test`
